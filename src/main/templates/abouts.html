    Персональный помощник для студентов


    Цель проекта

    Облегчить студентам получение знаний из зарубежных источников, затрачивая на эти задачи минимум времени.

    Функцианльность

    Перевод статьи с английского языка на русский и генерация краткого тезисного конспект текста.

    Лицензия

    Приложение включает в себя код из библиотеки Python, Django, Pandas, Numpy, Transformers, Hugging face, Bootstrap. Все компоненты входящие в состав приложения с открытым исходным кодом и имеют разрешающую лицензия GPL.
